# Java Hello Task

## Опис
Програма для курсу "Distributed Systems Seminars: Introduction".  
Вона:
- Запитує ім’я користувача та вітає його
- Рахує факторіал довжини імені
- Запитує дату народження та виводить вік у роках і днях
## Запуск
1. Переконайтеся, що у вас встановлений Java Development Kit (JDK)
   Це можна зробити командою - "javac -version"
2. У командному рядку (Windows CMD, PowerShell) виконайте:
   https://github.com/oleksiiopanashchukmtrim2025-ops/java-hello-task-lab2
   cd java-hello-task-lab2
3. У папці з файлом Main.java виконайте:
   javac Main.java
4. Запустіть програму командою:
   java Main.
